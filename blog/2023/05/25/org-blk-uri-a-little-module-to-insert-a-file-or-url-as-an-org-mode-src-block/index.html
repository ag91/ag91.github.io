org-blk-uri a little module to insert a file or url as an Org Mode src block

The other day I had an enjoyable EmacsBuddy videocall and, on the topic of "how do you use Org Mode", I showed without thinking how I capture file and web contents in my Org Mode files.

I usually use this to capture web pages that may be taken down (for example, job ads).

I then discovered that this feature could be of use also to others! So this afternoon I managed to release org-blk-uri, a tiny library on top of org-web-tools.
